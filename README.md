# Address Book Management System

A menu-driven Address Book Management System developed in C using file handling to efficiently store, manage, search, edit,delete, and organize contact information through a command-line interface.

---

# 📖 Project Overview

-> Managing personal or professional contacts manually can become difficult as the number of records increases. 

-> Traditional methods are time-consuming, less organized, and prone to data loss.

-> This project was developed to create a simple and efficient digital solution for storing and managing contact information using the C programming language.

-> The application allows users to perform operations such as adding, searching, editing, deleting, and viewing contacts while ensuring permanent storage using file handling techniques.

---

# 🎯 Problem Statement

Manual contact management systems have several limitations:

-> Difficulty in searching large numbers of contacts  
-> Duplicate or inconsistent records  
-> Lack of proper organization  
-> Risk of losing handwritten or temporary data  
-> Time-consuming process, while updating the contacts  

This project solves these problems by providing:

-> Fast contact searching  
-> Organized digital storage  
-> Easy modification and deletion of records  
-> Permanent data storage using files  
-> User-friendly menu-driven operations
